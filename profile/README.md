# Spire Studio

Spire Studio works on security, privacy, and trustworthiness in AI and data analytics.

We build research prototypes and developer tools for privacy-preserving AI systems, federated learning, and trustworthy data workflows.

## Projects

- **CloakBot**: a privacy-first AI assistant that sanitizes prompts locally before forwarding them to remote LLM APIs.
- **Figaro**: an intelligent federated learning platform for experiment orchestration, comparison, and analysis.

## Focus

- Privacy-preserving AI and data analytics
- Federated learning systems and tooling
- Trustworthy AI workflows
